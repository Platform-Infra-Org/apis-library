    FastAPI app API¶

    The application factory and shared infrastructure utilities.

    from tashtiot_apis_library import general_create_app
from tashtiot_apis_library.fastapi_template.utils import BaseAPI, settings

    Create and configure the FastAPI application.

    BaseAPI¶

    The outbound HTTP client wrapper the connectors build on. Accepts an httpx.Auth via auth=, so it composes with the SSO helpers.

    BaseAPI ¶

    BaseAPI(
    base_url: str,
    headers: Optional[Dict[str, str]] = None,
    auth: Optional[Union[Tuple[str, str], Auth]] = None,
    timeout: float = 10.0,
    verify: bool = False,
)

    base_url instance-attribute ¶

    base_url = rstrip('/')

    headers instance-attribute ¶

    headers = headers or {}

    auth instance-attribute ¶

    auth = auth

    timeout instance-attribute ¶

    timeout = timeout

    verify instance-attribute ¶

    verify = verify

    client property ¶

    client: AsyncClient

    Return an AsyncClient.

    • Inside async with: returns the reusable client.
    • Outside: returns a temporary client you can async with.

    Settings¶

    settings is an instance of ApplicationSettings. Its fields are the environment variables documented in the Configuration reference.
