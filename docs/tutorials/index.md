# Tutorials

Tutorials are **learning-oriented**: they take you by the hand through a complete, working example
so you can get a feel for the library. They don't aim to cover every option — that's what the
[How-to guides](../how-to/index.md) and [Reference](../reference/index.md) are for.

Work through them in order:

1. **[Your first app](first-app.md)** — install the library, create a FastAPI app with
   `general_create_app`, run it, and explore the batteries that come included (root route, metrics,
   health probes, Swagger).
2. **[Secure your API](secure-your-api.md)** — generate dev key material, turn on inbound JWT
   authentication, protect a route, and call it with a token via the Swagger **Authorize** dialog.

By the end you'll have a running, authenticated service and know where to look when you want to go
deeper.
